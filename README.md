# 생수가 흐르는 교회 찬양집
> 이 프로젝트는 생수가 흐르는 교회 찬양집 프로젝트입니다.

## Setup & Run
```
1. git clone https://github.com/livingwaterchurch0508/worship-next15.git
2. yarn
3. yarn dev
```

## Specification
```
node : 20.9.0
next.js : 15.0.1
react : ^19
```

## Dependencies
```
shadcn // UI 라이브러리
tailwindcss : 3.4.1 // css
lucide-react : 0.453.0 // 아이콘
next-themes : 0.3.0// 테마
zustand : 5.0.0 // 상태관리
```

## Deploy
```
yarn build
yarn start
```
